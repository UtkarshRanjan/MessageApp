# Photos Folder

Place your photos in this folder and add their filenames to the `config.js` file.

## How to Add Photos:

1. **Copy your photos** into this `photos` folder
2. **Edit `config.js`** and add the filenames to the `photos` array

### Example:

If you have these files in the photos folder:
- `birthday.jpg`
- `celebration.png`
- `memories.jpg`

Update `config.js` like this:

```javascript
photos: [
    "birthday.jpg",
    "celebration.png",
    "memories.jpg"
]
```

## Supported Formats:
- JPG/JPEG
- PNG
- GIF
- WebP
- SVG

The photos will automatically appear in the carousel on the message page!
